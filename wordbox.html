Project Goal

Design a web app (both front-end and back-end) for a target audience that is different than me.

Project Description

Worked in a team of 2 people and had about 8 weeks to complete the project. Given the time constraints and small group, we did not have the resources to conduct adequate user research or user testing for this project. Therefore, we decided to focus mainly on the coding aspect of the project.

The target audience we chose is nonverbal children, and the app would allow them to create sentences with the assistance of visual images and a text to speech functionality. We developed our app using HTML, CSS, and Javascript on a local server, and used Google Firebase for our real time database. Each week we iterated our design and added more functionality to our web app, including the use of an API to load images into the database based on the words and categories we curated.

My Contributions

  • created storyboard and initial wireframes for main pages of website
  • UI for Home Page and concept for Dashboard (main categories page)
  • set up API function calls to populate database with corresponding images
  • worked on drag & drop functionality
  • UI( CSS, structural, JS) improvements
  • created demo video (app walkthrough)
  • milestone write-ups

Project Outcome

We successfully completed our project on time and delivered a fully functional web app. Our final presentation was well received and our final project grade was A+.

What I Learned

In this project, I learned how to work with a database and integrate an API into our app. Working on both the frontend and backend helped me have a better understanding of how those two components interact. I also gained perspective on how to work effectively with a project leader and what the requirements are for an effective project or team lead.
